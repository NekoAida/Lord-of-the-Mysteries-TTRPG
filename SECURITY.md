# Security Policy

Thank you for helping keep the Lord of the Mysteries TTRPG system secure.

## Reporting a Vulnerability

If you believe you have found a security vulnerability, **please do not open a public GitHub issue.**

Instead, report it privately by:

- Opening a [GitHub Security Advisory](https://github.com/NekoAida/Lord-of-the-Mysteries-TTRPG/security/advisories/new) (if enabled), or
- Contacting the maintainer directly.

Please include:

- A description of the vulnerability.
- Steps to reproduce the issue.
- The potential impact.
- Any suggested fix, if available.

I will acknowledge reports as soon as reasonably possible and work to resolve confirmed vulnerabilities before public disclosure.

## Scope

This policy applies to:

- The Lord of the Mysteries TTRPG FoundryVTT system.
- Build scripts and development tooling contained in this repository.

It **does not** apply to:

- Foundry Virtual Tabletop itself.
- Third-party modules or dependencies maintained by others.

## Supported Versions

Only the latest released version is actively supported with security fixes.

| Version | Supported          |
|---------|--------------------|
| Latest  | ✅ Yes             |
| Older   | ❌ No              |
