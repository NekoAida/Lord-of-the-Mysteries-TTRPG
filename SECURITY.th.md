# นโยบายความปลอดภัย

ขอบคุณที่ช่วยรักษาความปลอดภัยของระบบ Lord of the Mysteries TTRPG

## การรายงานช่องโหว่

หากคุณเชื่อว่าพบช่องโหว่ด้านความปลอดภัย **กรุณาอย่าเปิด Issue สาธารณะบน GitHub**

ให้รายงานแบบส่วนตัวโดย:

- เปิด [GitHub Security Advisory](https://github.com/NekoAida/Lord-of-the-Mysteries-TTRPG/security/advisories/new) (หากเปิดใช้งาน) หรือ
- ติดต่อผู้ดูแลระบบโดยตรง

กรุณาระบุ:

- คำอธิบายช่องโหว่
- ขั้นตอนในการทำซ้ำปัญหา
- ผลกระทบที่อาจเกิดขึ้น
- วิธีแก้ไขที่แนะนำ (ถ้ามี)

ผมจะตอบรับรายงานโดยเร็วที่สุดเท่าที่เป็นไปได้ และจะแก้ไขช่องโหว่ที่ยืนยันแล้วก่อนเปิดเผยต่อสาธารณะ

## ขอบเขต

นโยบายนี้ใช้กับ:

- ระบบ Lord of the Mysteries TTRPG FoundryVTT
- สคริปต์สร้างและเครื่องมือพัฒนาที่อยู่ใน repository นี้

**ไม่** ใช้กับ:

- Foundry Virtual Tabletop เอง
- โมดูลหรือ dependency ของบุคคลที่สามที่ดูแลโดยผู้อื่น

## เวอร์ชันที่รองรับ

เฉพาะเวอร์ชันล่าสุดที่ปล่อยเท่านั้นที่ได้รับการสนับสนุนด้านการแก้ไขความปลอดภัย

| เวอร์ชัน   | รองรับ             |
|------------|-------------------|
| ล่าสุด     | ✅ ใช่             |
| เก่ากว่า   | ❌ ไม่             |
