# Lord of the Mysteries TTRPG — ระบบ FoundryVTT

**🌐 ภาษา / Language:** [ไทย](#คุณสมบัติ) ・ [English](README.md)

ระบบ FoundryVTT ที่สร้างขึ้นเองสำหรับ Lord of the Mysteries TTRPG นำโลกของ Beyonder, Sequence และ Pathway จากผลงานของ Cuttlefish That Loves Diving มาสู่โต๊ะเกมเสมือนจริง สร้างและดูแลโดย **NekoAida**

## คุณสมบัติ

- **ชีทตัวละคร Beyonder** — ชีทตัวละครเต็มรูปแบบสำหรับผู้เล่น พร้อมแท็บนำทาง (หลัก, ทักษะ, ของใช้, พลัง, ประวัติ)
- **คุณลักษณะหลัก 6 อย่าง** — ร่างกาย, ความคล่องตัว, สติปัญญา, การรับรู้, พลังใจ และ บารมี พร้อมคำนวณค่าปรับแต่งอัตโนมัติ
- **ทักษะที่ทอยได้ 13 อย่าง** — กรีฑา, ต่อสู้ระยะประชิด, อาวุธปืน, ลอบเร้น, มือไว, สืบสวน, เวทมนตร์, ประวัติศาสตร์, สังเกตการณ์, หยั่งรู้, เอาตัวรอด, การแสดง และ ชักจูง — แต่ละทักษะผูกกับคุณลักษณะที่ควบคุม
- **ทอยคุณลักษณะ & Saving Throw** — d20 + ค่าปรับแต่ง พร้อมรองรับโบนัสความชำนาญ, คริติคอล และ ฟัมเบิล
- **ระบบ Push** — ทอยใหม่เมื่อล้มเหลว โดยเสี่ยงได้รับค่าความบ้า
- **ระบบความบ้า (Madness)** — ติดตามตั้งแต่ 0–100 พร้อมทริกเกอร์อาการบ้าอัตโนมัติที่จุดสำคัญ (25, 50, 75) และเมื่อเพิ่มขึ้นมาก (5+) สูงสุดที่ Loss of Control เมื่อถึง 100
- **ติดตามจิตวิญญาณ (Spirituality)** — แถบทรัพยากรสำหรับพลัง Beyonder พร้อมค่าใช้จ่ายต่อความสามารถ
- **HP & สัญญาณชีพ** — พลังชีวิตพร้อมแถบแสดงเปอร์เซ็นต์
- **ความก้าวหน้า Sequence** — ตัวละครก้าวหน้าจากมนุษย์ธรรมดาผ่าน Sequence 9 ลงไปจนถึง Sequence 0
- **22 Pathway แห่งทวยเทพ** — The Fool, The Door, The Error, The Visionary, The Hanged Man, The Sun, The Tyrant, The White Tower และอื่นๆ — จัดระเบียบด้วยโฟลเดอร์ย่อยสำหรับ Pathway Powers และ สูตรยา & ส่วนผสม
- **ระบบอุปกรณ์** — ชีทอาวุธ, เกราะ และ ของใช้ พร้อมติดตามน้ำหนักและสลับการสวมใส่
- **ภาระบรรทุก (Encumbrance)** — คำนวณอัตโนมัติจากน้ำหนักของที่พกกับความจุตามค่าร่างกาย
- **พลัง Pathway** — ไอเทมสำหรับความสามารถ Beyonder พร้อมข้อกำหนด Sequence และค่าจิตวิญญาณ
- **สูตรยา (Potion Formula)** — ไอเทมสำหรับบันทึกส่วนผสมยาและ Sequence ที่เกี่ยวข้อง
- **สิ่งประดิษฐ์ผนึก (Sealed Artifact)** — ไอเทมเฉพาะสำหรับวัตถุลึกลับอันตราย
- **เชื้อสาย & ภูมิหลัง** — ลากและวางจากคลังข้อมูลเพื่อให้ความชำนาญทักษะและกำหนดตัวตนตัวละคร
- **ตัวสร้างตัวละคร (Character Builder)** — วิซาร์ดสร้างตัวละครที่ดึงข้อมูลจากคลังเชื้อสายและภูมิหลัง พร้อมให้ผู้เล่นจัดสรรค่าคุณลักษณะ
- **ระบบความชำนาญ** — โบนัสความชำนาญใช้กับทักษะและ save ให้ได้ผ่านเชื้อสาย, ภูมิหลัง หรือสลับด้วยตนเอง
- **สกุลเงิน** — ติดตาม Pound, Soli และ Pence บนชีทตัวละคร
- **การ์ดแชทพิเศษ** — เทมเพลต Handlebars สำหรับผลทอยลูกเต๋า พร้อมไฮไลท์คริติคอล/ฟัมเบิล และปุ่ม Push
- **คลังข้อมูล (Compendium)** — คลังข้อมูลสำเร็จรูปสำหรับ Pathway, เชื้อสาย และ ภูมิหลัง พร้อมสคริปต์มาโครสำหรับสร้างเป็นชุด
- **รองรับหลายภาษา** — รองรับภาษาอังกฤษและภาษาไทย

## การติดตั้ง

### ติดตั้งแบบ Manual

1. ใน Foundry VTT ไปที่ **Game Systems → Install System**
2. วางลิงก์ manifest URL ต่อไปนี้:
   ```
   https://github.com/NekoAida/Lord-of-the-Mysteries-TTRPG/releases/latest/download/system.json
   ```
3. คลิก **Install**

## ภาพรวมระบบ

### ประเภท Actor

| ประเภท    | รายละเอียด                           |
|-----------|--------------------------------------|
| Beyonder  | ชีทตัวละครผู้เล่น / NPC Beyonder      |

### ประเภทไอเทม

| ประเภท          | รายละเอียด                                         |
|-----------------|-----------------------------------------------------|
| Weapon          | อาวุธระยะประชิดและระยะไกล พร้อมน้ำหนักและสถานะสวมใส่ |
| Armor           | เกราะป้องกัน พร้อมน้ำหนักและสลับการสวมใส่            |
| Gear            | อุปกรณ์ทั่วไป พร้อมน้ำหนักและจำนวน                   |
| Pathway Power   | ความสามารถ Beyonder ผูกกับ Sequence และ Pathway       |
| Potion Formula  | สูตรยาเลื่อนขั้น พร้อมรายการส่วนผสม                 |
| Sealed Artifact | วัตถุลึกลับอันตรายและสิ่งของต้องสาป                  |
| Ancestry        | เชื้อสายตัวละคร ให้ความชำนาญทักษะ                    |
| Background      | ภูมิหลังตัวละคร ให้ความชำนาญทักษะ                    |

## โครงสร้างซอร์สโค้ด

```
css/                    # สไตล์ชีท
└── lotm.css                    # สไตล์ชีทหลักของระบบ
lang/                   # ไฟล์แปลภาษา
├── en.json                     # ข้อความภาษาอังกฤษ
└── th.json                     # ข้อความภาษาไทย
macros/                 # มาโครสร้างคลังข้อมูล
├── create-ancestries.js        # สร้างรายการเชื้อสายเป็นชุด
├── create-backgrounds.js       # สร้างรายการภูมิหลังเป็นชุด
└── create-pathways.js          # สร้าง 22 โฟลเดอร์ Pathway และไอเทมตั้งต้น
module/                 # โมดูล JS ที่กำหนดลอจิกของระบบ
├── apps/                       # หน้าต่างแอปพลิเคชัน
│   └── character-builder.mjs           # วิซาร์ดสร้างตัวละคร
├── data/                       # คำจำกัดความ TypeDataModel
│   ├── actor-beyonder.mjs              # สกีมาข้อมูล Beyonder (คุณลักษณะ, ทักษะ, สกุลเงิน, ความบ้า)
│   ├── item-ancestry.mjs               # สกีมาข้อมูลเชื้อสาย
│   ├── item-armor.mjs                  # สกีมาข้อมูลเกราะ
│   ├── item-background.mjs            # สกีมาข้อมูลภูมิหลัง
│   ├── item-gear.mjs                   # สกีมาข้อมูลของใช้
│   ├── item-pathway-power.mjs          # สกีมาข้อมูลพลัง Pathway
│   ├── item-potion-formula.mjs         # สกีมาข้อมูลสูตรยา
│   ├── item-sealed-artifact.mjs        # สกีมาข้อมูลสิ่งประดิษฐ์ผนึก
│   └── item-weapon.mjs                 # สกีมาข้อมูลอาวุธ
├── documents/                  # คลาสเอกสารแบบ Override
│   └── actor.mjs                       # LOTMActor — ข้อมูลอนุพันธ์, ทอยลูกเต๋า, ลอจิกความบ้า
├── sheets/                     # คลาสชีท
│   ├── actor-sheet.mjs                 # LOTMActorSheet — ชีทตัวละคร Beyonder
│   └── item-sheet.mjs                  # LOTMItemSheet — ชีทไอเทมรวม
└── lotm.mjs                    # จุดเริ่มต้นหลัก — ลงทะเบียนโมเดล, ชีท และ hooks
scripts/                # สคริปต์ยูทิลิตี้
└── macros/                     # สคริปต์มาโครในเกม
    ├── create-fool-pathway.js          # สร้างข้อมูล Pathway The Fool
    └── create-seer-items.js            # สร้างไอเทม Sequence 9 Seer
templates/              # เทมเพลต HTML แบบ Handlebars
├── actor/                      # เทมเพลตชีท Actor
│   └── actor-sheet.hbs                 # เลย์เอาต์ชีทตัวละคร Beyonder
├── apps/                       # เทมเพลตแอปพลิเคชัน
│   └── character-builder.hbs           # ฟอร์มตัวสร้างตัวละคร
├── chat/                       # เทมเพลตข้อความแชท
│   └── custom-roll.hbs                 # การ์ดผลทอยลูกเต๋าแบบกำหนดเอง
└── item/                       # เทมเพลตชีทไอเทม
    └── item-sheet.hbs                  # เลย์เอาต์ชีทไอเทมรวม
system.json             # ไฟล์ manifest ระบบ FoundryVTT
template.json           # คำจำกัดความประเภท Actor & Item
```
